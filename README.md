# Shell Calculadora - Calculadora Simples em Bash

Este é um script em Bash que oferece funcionalidades básicas de uma calculadora. Ele permite realizar operações simples de adição, subtração, multiplicação e divisão.

## Como usar

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/handyman0/shell_calculadora.git
   ```

2. **Execute o script:**

   ```bash
   cd shell_calculadora
   chmod +x shell_calculadora.sh
   ./shell_calculadora.sh
   ```

3. **Siga as instruções:**

   O script solicitará dois valores e uma escolha de operação. Escolha a operação desejada digitando o número correspondente ou 'Q' para sair.

## Observações

- Certifique-se de ter o Bash instalado para executar o script.
